Accountability: About This Data Set

The data set I collected through the past two weeks include location and battery usage. In the first graph, it shows the places I've been to in Pittsburgh on a map, mainly including CMU main campus, Sterling Plaza and my home. In the second graph, it shows the battery usage of my iPhone with a relatively rapid scanning speed. We can see that for most of the time, my battery is in a low battery storage level.

Where did this data come from?

It is collected by an APP called Aware, stored in my iPhone.

Who has access to it?

Initially only I could get access to the data. As long as I post it publicly, everyone knowing the website url could get access to it.
